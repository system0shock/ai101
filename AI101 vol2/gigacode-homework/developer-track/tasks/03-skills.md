# 03. Skills для разработчика

## Цель

Использовать проектный Skill для проверки маленького изменения.

## Запрос

```text
Используй Skill code-review и проверь мое исправление DiscountPolicy.
```

## Ожидаемый результат

Ответ должен содержать:

- замечания;
- доказательства;
- команду проверки;
- остаточный риск.

## Самопроверка

- Skill не предлагает большой рефакторинг.
- Проверка ссылается на тест или код.
