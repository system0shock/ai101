# 01. Commands для разработчика

## Цель

Попробовать `@`, `!` и пользовательскую команду на кодовой задаче.

## Шаги

1. Добавьте стандарт в контекст:

```text
@developer-track/docs/code-standards.md
```

2. Добавьте README режима:

```text
@developer-track/README.md
```

3. Запустите проверку через команду оболочки:

```text
!cd developer-track && ./scripts/run-tests.sh
```

На Windows используйте:

```text
!cd developer-track; .\scripts\run-tests.ps1
```

4. Вызовите пользовательскую команду:

```text
/summarize-context
```

## Ожидаемый результат

Агент должен увидеть, что есть один падающий тест, и не предлагать массовое переписывание.

## Самопроверка

- В контексте есть `code-standards.md`.
- Есть вывод тестов.
- Сводка отделяет факты от предположений.
