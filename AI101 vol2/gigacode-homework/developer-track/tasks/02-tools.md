# 02. Tools

Цель: пройти цикл explore -> read -> edit -> verify.

## Базовое задание

Попросите агента найти причину падения теста: сначала проверка, затем поиск, чтение нужных файлов, минимальная правка и повторный тест.

## Самостоятельно

Напишите prompt, который явно просит использовать поиск файлов, чтение файла, редактирование и shell-проверку. В конце попросите tool-use summary.
