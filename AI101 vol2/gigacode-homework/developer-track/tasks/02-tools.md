# 02. Tools

Цель: пройти цикл explore -> read -> edit -> verify.

## Конкретная цель
<!-- concrete-goal -->

Починить падающий тест `appliesBulkDiscountFromTenItems` минимальной правкой в коде скидки и получить проходящий targeted test.

## Документация
<!-- docs-links -->

- Official Qwen Code Tools: https://qwenlm.github.io/qwen-code-docs/en/tools/index
- Локальная справка: `../../.gigacode/skills/qwen-code-helper/references/commands-tools.md`

## Базовое задание

Попросите агента найти причину падения теста: сначала проверка, затем поиск, чтение нужных файлов, минимальная правка и повторный тест.

## Самостоятельно

Напишите prompt, который явно просит использовать поиск файлов, чтение файла, редактирование и shell-проверку. В конце попросите tool-use summary.

## Подсказка для самостоятельной части
<!-- self-hint -->

Соберите prompt как чек-лист: "сначала запусти тест", "найди класс через поиск", "прочитай только нужные файлы", "предложи правку", "внеси минимальное изменение", "повтори тест", "перечисли tool calls". Успех: тест проходит, а tool-use summary показывает search/read/edit/shell этапы.
