# 06. Subagents для разработчика

## Цель

Делегировать узкую проверку именованному агенту.

## Запрос

```text
Запусти именованного агента test-investigator для расследования падающего теста. Передай ему только developer-track и попроси не менять файлы.
```

## Ожидаемый результат

Именованный агент должен вернуть расследование только для чтения.

## Самопроверка

- Именованный агент не меняет файлы.
- Ответ объясняет, что именованный агент блокирует родительский диалог до завершения.
- Ответ объясняет, что именованный агент стартует без истории родительского диалога, поэтому ему нужен явный контекст.
