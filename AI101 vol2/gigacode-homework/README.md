# Домашнее задание AI101 vol.2: GigaCode tools

Практический комплект к лекции про команды, tools, skills, hooks, MCP и субагентов.

## Как проходить

1. Откройте терминал в `gigacode-homework/`.
2. Запустите GigaCode CLI.
3. Выберите трек: `developer-track/` или `analyst-track/`.
4. Идите по заданиям `tasks/01-commands.md` ... `tasks/06-subagents.md`.

Fork-субагенты в этом комплекте отключены. Для делегирования используйте только именованных subagents через `subagent_type`.

Если непонятно, какой механизм использовать или почему не работает пример, попросите GigaCode применить skill `qwen-code-helper`.
