# Expected stakeholder summary

Разделяет подтверждённые требования, конфликты, вопросы и риски публичных обещаний.

Пример структуры:

**Подтверждено:**
- Зарплатный сегмент — операции готовы.
- Срок до 3 рабочих дней — юристы зафиксировали.

**Конфликты:**
- Маркетинг: «1 час для всех» — не согласовано с PRD и юристами.
- Лимит: 300k (риски) vs 500k (PRD) vs 1000k (FAQ).
- Ставка: 9,9% (FAQ) vs 14,9% (PRD).

**Открытые вопросы:**
- Какой лимит утверждён для старта?
- Промо-ставка 9,9% — существует ли она и для кого?
- Кто согласует итоговый FAQ перед публикацией?
