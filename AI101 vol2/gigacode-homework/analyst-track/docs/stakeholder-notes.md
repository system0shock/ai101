# Stakeholder notes

Маркетинг хочет обещать одобрение за 1 час для всех клиентов банка.

Риски готовы стартовать только с лимитом 300 000 руб. на первом этапе.

Юристы требуют указывать реальный срок «до 3 рабочих дней», не «мгновенно» и не «за 1 час».

Операции подтвердили готовность только для зарплатного сегмента.
