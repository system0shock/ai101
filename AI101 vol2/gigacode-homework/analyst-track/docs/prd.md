# PRD: онлайн-кредит «Быстрый старт»

Цель: запустить онлайн-подачу заявки на потребительский кредит через мобильное приложение.

Ограничения:

- Запуск только для клиентов с зарплатным счётом в банке (от 6 месяцев).
- Максимальная сумма: 500 000 руб.
- Срок рассмотрения: до 3 рабочих дней.
- Ставка: от 14,9% годовых (зависит от кредитного рейтинга клиента).
- Подача заявки — только через мобильное приложение.
