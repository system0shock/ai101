# 05. MCP

Цель: спроектировать read-only источник знаний.

## Базовое задание

Попросите агента объяснить `shared/mcp/local-knowledge-server.example.json` с точки зрения аналитика.

## Самостоятельно

Спроектируйте MCP tool `mcp__knowledge__search_policy`: параметры поиска, read-only гарантия, пример ответа и exclude для write-tools.
