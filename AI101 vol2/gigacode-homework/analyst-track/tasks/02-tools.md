# 02. Инструменты для аналитика

## Цель

Попросить агента найти противоречия между документами.

## Запрос

```text
@analyst-track/docs
Найди противоречия между PRD, FAQ, заметками к релизу и заметками заинтересованных сторон. Сначала покажи доказательства по файлам, затем предложи вопросы к владельцу продукта.
```

## Самопроверка

- Агент сравнил несколько документов.
- Ответ содержит конкретные пары конфликтующих утверждений.
- Агент не переписал PRD до согласования решений.
