# 02. Tools

Цель: увидеть, как агент читает документы и ищет противоречия.

## Базовое задание

Попросите агента найти противоречия между `prd.md`, `stakeholder-notes.md`, `faq.md` и `release-notes.md` с file references.

## Самостоятельно

Напишите prompt, который явно просит поиск по документам, чтение файлов, разрешенное редактирование черновика и shell-команду для списка файлов. В конце попросите tool-use summary.
