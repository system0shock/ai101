# 02. Tools

Цель: увидеть, как агент читает документы и ищет противоречия.

## Конкретная цель
<!-- concrete-goal -->

Получить contradiction report по онлайн-кредиту «Быстрый старт»: минимум три конфликта между PRD, stakeholder notes, FAQ и release notes, каждый с file reference и предложенным вопросом владельцу.

## Документация
<!-- docs-links -->

- Official Qwen Code Tools: https://qwenlm.github.io/qwen-code-docs/en/tools/index
- Локальная справка: `../../.gigacode/skills/qwen-code-helper/references/commands-tools.md`

## Базовое задание

Попросите агента найти противоречия между `prd.md`, `stakeholder-notes.md`, `faq.md` и `release-notes.md` с file references.

## Самостоятельно

Напишите prompt, который явно просит поиск по документам, чтение файлов, разрешённое редактирование черновика и shell-команду для списка файлов. В конце попросите tool-use summary.

## Подсказка для самостоятельной части
<!-- self-hint -->

Разделите prompt на этапы: "покажи список документов", "прочитай только релевантные", "найди противоречия с источниками", "если я разрешу, предложи правку черновика". Успех: отчёт содержит конфликты "любой клиент vs зарплатный сегмент", "1 час vs 3 рабочих дня", "9,9% vs 14,9%" и "1 000 000 vs 500 000 руб.".
