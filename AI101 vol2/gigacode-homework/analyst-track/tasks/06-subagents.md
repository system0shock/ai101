# 06. Subagents для аналитика

## Цель

Делегировать поиск противоречий именованному агенту.

## Запрос

```text
Запусти именованный агент consistency-reviewer для analyst-track/docs. Передай ему задачу найти противоречия по формату, роли, периоду, хранению и уведомлениям.
```

## Ожидаемый результат

Агент возвращает список противоречий с доказательствами.

## Самопроверка

- Именованный агент получил явную область анализа.
- Ответ не предполагает, что именованный агент видел историю родительского диалога.
- Fork subagent упомянут как отдельный режим, если ваша версия CLI его поддерживает.
